# Spending Tracker

A Django-based personal finance management web application designed to help users efficiently manage their income and expenses. The application provides an intuitive dashboard for tracking financial activities, organizing transactions by categories, and monitoring overall financial health.

## Features

* User Registration and Authentication
* Secure Login and Logout System
* Personalized Dashboard
* Add, Edit, and Delete Transactions
* Income and Expense Tracking
* Transaction Categorization
* Financial Summary and Balance Calculation
* Transaction History Management
* Responsive and User-Friendly Interface

## Technology Stack

### Backend

* Python
* Django

### Frontend

* HTML5
* CSS3
* JavaScript
* Bootstrap

### Database

* SQLite

## Installation

### Clone the Repository

```bash
git clone https://github.com/Afeefaaa12/spending_tracker.git
cd spending_tracker
```

### Create and Activate Virtual Environment

#### Windows

```bash
python -m venv venv
venv\Scripts\activate
```

#### Linux / macOS

```bash
python3 -m venv venv
source venv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Apply Migrations

```bash
python manage.py makemigrations
python manage.py migrate
```

### Run the Development Server

```bash
python manage.py runserver
```

Open your browser and visit:

```text
http://127.0.0.1:8000/
```
